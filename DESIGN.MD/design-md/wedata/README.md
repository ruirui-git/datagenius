# WeData Design System

`DESIGN.md` 基于 WeData 3.0 的 Figma 设计稿和 TDesign 主题 token 提炼而成。覆盖品牌色、排版、组件、布局、深度等全部设计维度，供 AI Agent（Claude、Cursor 等）生成符合 WeData 视觉语言的 UI。

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | 完整设计系统文档（9 章） |
| `preview.html` | 交互式设计 token 目录（浅色） |
| `preview-dark.html` | 交互式设计 token 目录（深色）— WeData 当前为浅色系统，此文件为占位 |

## 设计签名

- **墨色四阶** — `rgba(0,0,0, 0.9/0.7/0.5/0.3)` 四级透明度层次体系
- **胶囊统一律** — 所有交互控件共享 100px 圆角
