## Case 1：物流时效分析与告警

**文件**：`case1-物流时效分析与告警.html`

**用户提问**：
> 我们物流经常被客户投诉太慢，帮我分析一下各个州和各个卖家的配送时效情况。如果某个州的配送超过15天或某个卖家的准时率低于80%，帮我盯着并告警。另外帮我归因一下，物流慢到底是卖家发货慢还是物流运输慢。

**Agent 回答摘要**：
- 平均配送 **18.8 天**，其中运输阶段占 82%（15.5 天），是投诉的核心痛点
- **23/27 个州**平均配送超 15 天阈值，2 个统计异常州（RR 29.7d、SP 8.7d）
- **44 个卖家**准时率低于 80%
- 归因结论：物流慢主因是运输网络瓶颈（82%），而非卖家发货效率（15%）

**产物包含**：
- 4 个 Plotly 交互式图表（州配送时效柱状图、卖家准时率分布、月度趋势、归因瀑布图）
- 4 个 `.insight` 分析区块（含 L1-L4 深度洞察）
- SQL 折叠查看按钮
- 导出 PDF/DOCX 按钮（inject_export 注入）

**产物类型**：异动洞察报告（`anomaly-insight.md` 规范）

## Case 2：客户流失预警与营销

**文件**：`case2-客户流失预警与营销.html`

**用户提问**：
> 帮我分析一下我们的客户情况，查一下哪些客户快要流失了、哪些是高价值客户。我想每周收到一份报告：高风险流失客户有多少、他们之前买了什么品类、平均消费多少。然后针对不同客户群体给我营销建议，比如高价值客户发优惠券、沉默客户发召回短信。

**Agent 回答摘要**：
- 23,653 个客户，100% 仅购买 1 次（F=1），改用 R×M 双维度分群
- **高风险流失 5,892 人（24.9%）**，其中高价值流失 1,468 人，潜在损失 R$509k
- 8 个客户群体：高价值活跃（6.1%）、高价值沉默（12.2%）、高价值流失（6.2%）、潜力客户（12.6%）等
- 营销建议：高价值流失→15% 无门槛券召回、高价值活跃→交叉销售、潜力客户→满减升级

**产物包含**：
- 3 个 Plotly 图表（客户分群散点图、分群分布柱状图、品类偏好热力图）
- 分群统计表格
- 营销策略建议
- SQL 折叠按钮 + 导出按钮

**产物类型**：智能分析报告（`intelligent-report.md` 规范）

## Case 3：月度销售额问数+分析

**文件**：`case3-月度销售额问数分析.html`

**用户提问**：
> 各个月的销售额是多少，哪些月份销售额表现较好，哪些较差

**Agent 回答摘要**：
- 21 个月数据（2016-10 ~ 2018-08），整体趋势"快速爬坡→高位震荡"
- 表现最好：2017-11（R$987k，Black Friday）、2018-05（R$977k）
- 表现最差：2017-01（R$112k，爬坡期）、2018-06~08 连续回落（-14%）
- 2018 下半年增长减速信号值得警惕

**产物包含**：
- 1 个 Plotly 双轴图表（销售额柱状 + 订单量折线，峰值/低谷高亮标注）
- SQL 折叠按钮 + 导出按钮
- 浅色页面模板（CSS 变量规范）

**产物类型**：智能问数（`intelligent-query.md` 规范）
